The following versions of Exponential Platform are currently receiving security updates:

| Version | Supported |
|---|---|
| 2.5.x | ✅ |
| 1.x.x | ❌ |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report security issues responsibly by emailing:  
**security@se7enx.com**

You can expect a response within **1 business day**. We will keep you updated as we investigate and address the issue. If the vulnerability is confirmed you will be credited in the security advisory (unless you prefer to remain anonymous).

## Security Advisories

Published security advisories for Exponential Platform are available via the GitHub Security Advisories tab on each affected repository.
